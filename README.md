# 🧮 Arbitrary Precision Calculator (APC)

A powerful **Arbitrary Precision Calculator** developed using **C Programming** and **Data Structures** to perform arithmetic operations on extremely large numbers that exceed standard data type limits.

---

# 📌 Project Overview

Traditional data types like `int`, `long`, and `long long` have storage limitations.

This project overcomes those limitations by implementing arithmetic operations using **Doubly Linked Lists**, enabling calculations on numbers with hundreds or even thousands of digits.

---

# 🚀 Features

✅ Addition of large numbers
✅ Subtraction of large numbers
✅ Multiplication support
✅ Division support
✅ Handles numbers beyond integer limits
✅ Dynamic memory allocation

# ⚙️ Core Concepts

## 🔹 Arbitrary Precision Arithmetic

Allows calculations on numbers larger than built-in C data types.

## 🔹 Doubly Linked List

Each digit is stored as a node in a doubly linked list for efficient traversal and arithmetic operations.

## 🔹 Dynamic Memory Management

Memory is allocated dynamically depending on input size.
